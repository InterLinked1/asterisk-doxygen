Detailed Description

Headerless G.726 (16/24/32/40kbps) data format for Asterisk.

File name extensions:

  • 40 kbps: g726-40
  • 32 kbps: g726-32
  • 24 kbps: g726-24
  • 16 kbps: g726-16
