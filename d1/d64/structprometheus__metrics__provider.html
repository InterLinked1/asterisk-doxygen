Detailed Description

A function table for a metrics provider.

It's generally nice to separate out things that provide metrics from the core of this module. For those that want to be notified when things happen in the core module, they can provide an instance of this function table using prometheus_metrics_provider_register and be notified when module affecting changes occur.
