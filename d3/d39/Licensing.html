Licensing of 3rd Party Code

This section contains a (not yet complete) list of libraries that are used by various parts of Asterisk, including related licensing information.

ALSA Library

  • Library: libasound
  • Website: http://www.alsa-project.org
  • Used by: chan_alsa
  • License: LGPL

OpenSSL

  • Library: libcrypto, libssl
  • Website: http://www.openssl.org
  • Used by: Asterisk core (TLS for manager and HTTP), res_crypto
  • License: Apache 2.0
  • Note: An exception has been granted to allow linking of OpenSSL with Asterisk.

Curl

  • Library: libcurl
  • Website: http://curl.haxx.se
  • Used by: func_curl, res_config_curl, res_curl
  • License: BSD

PortAudio

  • Library: libportaudio
  • Website: http://www.portaudio.com
  • Used by: chan_console
  • License: BSD
  • Note: Even though PortAudio is licensed under a BSD style license, PortAudio will make use of some audio interface, depending on how it was built. That audio interface may introduce additional licensing restrictions. On Linux, this would most commonly be ALSA: ALSA Library.
